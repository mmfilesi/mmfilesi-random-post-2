== Description ==

This plugin shows in the sidebar n post related with the same category(s) of post being viewed.
The result can be random or chronological order.

Este plugin es un widget que se incluye en el sidebar. Muestra n entradas relacionas con la entrada que se ha cargado.
Los resultados pueden ordenarse de forma aleatoria o por orden cronologico.

== Installation ==

1. Upload mmfilesi-random-post to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place mmfilesi-random-post in your sidebar
